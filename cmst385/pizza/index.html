Example form for ordering a pizza

This is a standard HTML form that utilizes with textfields, checkboxes, and radio buttons. Since we are using GitHub pages to host our website we will need a third party from submission service which will email the form to our provided email address. I recommend using https://formspree.io/ which is a free service as long as the submissions are limited to 1000 per month. Like who would want to receive more then 1000 emails per month, but if you do you can subscribe to their paid service for $10 per month. So lets get started...

WebJeda has a good 8 minute video tutorial about the process at:
https://youtu.be/IP6HsgwQkvs
