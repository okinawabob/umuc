PROJECT DESCRIPTIONS

Assignments: 

Assignments will be given throughout the term covering the material discussed in class the week before. Assignments must be submitted on the due dates or there will be a 30% point reduction for each class period late. No assignment will be accepted after the final exam time.

Grading will be 80% objective (results, explanations, conclusions) and 20% subjective (neatness, clarity, extra work). A project report that minimally meets all specifications will receive a score of 80% of the total points. If any portion of a project is plagiarized (Using some one’s code and saying it is your own), the entire project will receive a score of zero.

Assignment 1: HTML  (15%)

Assignment 2: Colors and Images (15%)

Assignment 3: CSS2(15%)

Website Project:

The purpose of the project is to demonstrate your ability to apply the skills that you have learned during the course by creating a fully functional web site using various tools and methodologies, including HTML5, Web graphics, and Cascading Style Sheets.

Website Proposal (5%)

You will create a website proposal, the main purpose of which is that you learn to conduct good project planning. Present this as a 5-10 minute presentation to the class using a PowerPoint slide show. Review and critique your fellow classmates' draft website proposals.

Your project can be a website designed for your workplace, nonprofit/volunteer work, or another business/organization, real or fictitious.  Hopefully, the project will be something of benefit that you can take away from this course.

Create your full Website (30%)

Your completed website must contain a minimum of five html web pages (not including a references page, if you use other authors' work with their permission), properly hand-coded by you to include standards compliant HTML5 and CSS3. Your website must be original work created during this term and cannot have been previously created for any other purpose.  Finally, you will review and critique fellow classmates' completed website projects.

Final Exam (20%)

The final exam will be comprehensive and cover topics discussed throughout the course.

Attendance:

Class attendance is mandatory and understanding each lesson depends on understanding the previous lesson.  The student is responsible for any work missed due to absence.
