Linear Equation: Slope Intercept Form

Change the slope (m) and y-intercept (b) by moving the sliders for both linear equations Line1 and Line 2.

Problem 1: Adjust sliders so that you have two parallel lines. Record the numerical values for slope and y-intercept for each line. Print the graph.

Problem 2: Adjust sliders so that you have two perpendicular lines. Record the numerical values for slope and y-intercept for each line. Print the graph. Determine and record the point of intersection for the two perpendicular lines.
