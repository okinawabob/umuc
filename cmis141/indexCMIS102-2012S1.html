Week 6:

The sixth week we will focus on Chapter 6 content covering data Arrays. They allows you to have multiple variables with the same name and different index numbers. This is analogous to a post office with different box numbers. Arrays are a powerful method for storing data that enables the use of loops to process this data. Arrays are the way to implement mathematical matrix concepts. You will need to implement arrays in your Final Assignment 5.

After the exam we will focus on Chapter 7 content covering Program Functions which allows you to break up your code into modules. Functions are defined with a function definition, which specifies what code will be executed when the function is called.
